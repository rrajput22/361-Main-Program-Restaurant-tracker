# Restaurant Tracker

Web app that lets users add, view, edit, and favorite restaurants.

## Run
Open index.html in a web browser.

## Features
- Add restaurants (name, location, visited, rating, notes)
- View and edit restaurant details
- Separate pages for Home, Add, View Details, Favourites, Help, and Welcome
- Confirmation pop-ups for saving and deleting entries
- Consistent navigation across all pages
