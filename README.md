# Restaurant Tracker

Single-page web app to add, edit, and favorite restaurants.
Data is stored locally in the browser using LocalStorage.

## Run
Open index.html in a web browser.

## Features
- Add restaurants (visited, rating, notes)
- Edit or delete entries
- Mark favourites and view them on the Favourites page
- Help page and consistent navigation menu
- Confirmation pop-ups for save and delete actions
